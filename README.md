## Fluxo de versionamento

```bash
git init
git add .
git commit -m "feat: descrição da alteração"
git push origin main
```

Commits seguem o padrão [Conventional Commits](https://www.conventionalcommits.org/): `feat`, `fix`, `docs`, `chore`.
